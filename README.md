## ⚙️ How To Use The App

Open the program after the download finishes. You see a simple dashboard. Drag a PDF file into the window to open it. Use the buttons on the top bar to perform tasks.

### Manage Pages
Click the "Pages" tab to see all thumbnails. Drag thumbnails to reorder pages. Right-click a thumbnail to delete or rotate the page. Use the "Split" button to save selected pages as a new file.

### Combine Documents
Select "Merge" from the file menu. Add all the files you want to join. Drag them into the sequence you prefer. Click "Create" to save your new file.

### Apply Signatures
Open the document you need to sign. Go to the "Tools" menu and select "Sign". Draw your signature or import an image file. Place the signature on the page. Save the document to lock the change.

## 🔍 Troubleshooting

**The app does not open.**
Make sure you have Windows 10 or newer. Check that your user account has permission to run programs in the folder where you placed the file.

**The file looks blurry.**
Check your screen settings. Ensure you use the native resolution of your monitor.

**The signature feature does not appear.**
Ensure you saved the document after you imported your signature. If the file is a read-only document, save a copy first, then sign that copy.

**The app runs slow.**
Close other programs that use high memory. PDF files with many images take more system resources to render.
